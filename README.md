# To-Do List Web App

A simple and responsive To-Do List web application built with HTML, CSS, and JavaScript. Tasks are saved in the browser using Local Storage, so they persist even after refreshing the page.

## Features
- Add new tasks
- Mark tasks as completed (with checkbox and strikethrough)
- Delete tasks
- Data persistence using Local Storage
- Clean and modern user interface
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

## How to Use
1. Open `(https://shantanuchakrabarty.github.io/todo-list-app/)` in your browser.
2. Type a task in the input field and click **Add**.
3. Check the checkbox to mark a task as completed.
4. Click the **Delete** button to remove a task.
5. Refresh the page — your tasks will still be there.
